
[Еще не] Полное описание команд, воспринимаемых контроллером монтировки AMOUNT

Оставляю за собой право изменить формат любой из команд без предупреждения :)

При описании команд
в одиночных кавычках - одиночные символы
в двойных кавычках - символьные строки, более 1 символа

 '?' - возвращает '#'
    
 '>' - движение монтировки вправо, со скоростью, указанной после '>', максимально ">9", остановка ">0", при правильном восприятии команды, возвращает '#'
 '<' - движение монтировки влево,   со скоростью, указанной после '<', максимально "<9", остановка "<0", при правильном восприятии команды, возвращает '#'
 '^'  - движение монтировки вверх,   со скоростью, указанной после '^',  максимально "^9", остановка "^0",  при правильном восприятии команды, возвращает '#'
 'V' - движение монтировки влево,   со скоростью, указанной после 'V', максимально "V9", остановка "V0", при правильном восприятии команды, возвращает '#'
    
 'a' - запрос привязана ли монтировка к текущим координатам, возвращает "1#", если привязана, и "0#", если не привязана
 'A' - принудительная привязка к текущим координатам, возвращает '#', если команда принята

 'd' - запрос даты из контроллера в формате "YYMMDD#", где YY - две последние цифры года, MM - месяц, DD - день, все значения с ведущими нулями, если значение меньше 10
 'D' - установка даты в контроллер  в формате "DYYMMDD", где YY - две последние цифры года, MM - месяц, DD - день, все значения с ведущими нулями, если значение меньше 10
    
 'e' - одна из двух, сохраненных для Стеллариума команда NexStar протокола, запрос текущих экваториальных координат
         
 'F' - движение фокусера со скоростью, указанной после символа направления '+' или '-', "F+9" - максимальная скорость, "F0" - остановка, при восприятии команды, возвращается '#'
    
 'G' - включение/отключение режима отладки, возвращает "UnDebug#" - если отладка отключена, "Debug#" - отладка включена

 'K' - (эхо) возвращает символ, следующий за символом 'K' + '#', "K5", вернет "5#"

 'm' - запрос типа монтировки, возвращает "0#", если тип монтировки неопределен, "1#" если монтировка альт-азимутальная, "2#" EQ Noth, "3#" EQ South
 'M' - принудительная установка типа монтировки в режим  "M1" - Альт-Азимутальный, "M2" - EQ Noth, "M3" - EQ South, при восприятии команды, возвращается '#'
          
 'n' - запрос активности моторов, возвращает "0#" - моторы отключены, "1#" - моторы включены  
 'N' - включение/отключение моторов "N0" - отключить, "N1" - включить, при восприятии команды возвращает '#'

 'O' - управление затвором фотоаппарата "OP,E" где P - количество кадров, E - время экспозиции в секундах, 
        например, "O32,50" - сделать 32 снимка по 50 сек. Межкадровый интервал задается в конфиге.
        "O32,0" - сделает 32 снимка с минимальной выдержкой, задаваемой в конфиге,
        эта возможность сделана для использования выдержек, определяемых фотоаппаратом,
        т.к. минимальная выдержка подает кратковременный сигнал на затвор.
        При восприятии команды возвращает '#'.

 'P' - команда гидирования Push. "P>1" - один микрошаг вправо, "P<2" - два микрошага влево
        "P^3" - три микрошага вверх, "PV5" - пять микрошагов вниз.
        При восприятии команды возвращает '#'.

 'r' - вторая сохраненная для Стеллариума команда NexStar, позиционирование на указанные экваториальные координаты
            
 's' - запрос скорости ведения (трекинг) монтировки; Возвращает "0#" - остановлена, "1#" - звездная скорость,
        "S2" - солнечная скорость, "S3" - лунная скорость.

 'S' - установка скорости ведения монтировки: "S0" - отключить ведение, "S1" - ведение со звездной скоростью,
        "S2" - ведение с солнечной скоростью, "S3" - ведение с лунной скоростью.
       При восприятии команды возвращает '#'.
               
 't' - запрос времени монтировки, ответ в формате "ЧЧММСС#"

 'T' - установка времени  монтировки в формате "TЧЧММСС#"
                         
 'x' - запрос координаты X монтировки, ответ поступает в долях окружности [0,1], например "0.0416667#" соответствует 15 градусам азимута или 1 часу прямого восхождения
        при этом, экваториальная монтировка возвращает прямое восхождение, азимутальная - азимут.

 'X' - установка координаты X, задаваемой в долях окружности [0,1], например "X0.0416667" соответствует 15 градусам азимута или 1 часу прямого восхождения
        при этом, экваториальная монтировка воспринимает прямое восхождение, азимутальная - азимут.
        При восприятии команды возвращает '#'.

 'y' - запрос координаты Y монтировки, ответ поступает в долях окружности [0,1], например "0.0416667#" соответствует 15 градусам высоты или склонения
        при этом, экваториальная монтировка возвращает склонение, азимутальная - высоту.

 'Y' - установка координаты Y, задаваемой в долях окружности [0,1], например "Y0.0416667" соответствует 15 градусам высоты или склонения
        при этом, экваториальная монтировка воспринимает склонение, азимутальная - высоту.
        При восприятии команды возвращает '#'.
